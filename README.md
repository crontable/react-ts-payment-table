# 주요 설계

- 주요 설계 의도와 트레이드오프 2~3개

1. 도메인을 먼저 파악하면서 데이터 렌더링
2. 시안대로 페이지로 구현
3. 컴포넌트 쪼개기
4. 검색 컴포넌트 + 검색 기능
5. 자잘한 스타일링 및 미비된 부분

# 요구 사항

## Ordered 영역

- [x] consumption 항목을 그룹핑하여 Sub.TTL 계산된 수치를 출력한다.
- [x] consumption 항목의 모든 총합(G.TTL)의 계산된 수치를 출력한다.

## Payable 영역

- 요구사항 체크리스트
